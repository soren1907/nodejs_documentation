Tools


Nodemon

It is a tool that listens to file changes and restarts the app when a change occours.

It should only be used for development and not for production.

It can be installed globally, which means outside the project so that it can be used everywhere, or it can be installed locally which means related to this unique project.


Postman

It is really useful when it comes to testing API’s. It acts as an client and can test the application.

It makes it possible to send POST, PATCH and DELETE requests to a server with data stored in the request body.

You can choose the type of method and send data in the request bodt by going to:

Body -> raw, and change “text” to “json”

It can present the data in a prettier format and can also be used for sending query parameters: “/?key=value” or path variables: “/users/1”
