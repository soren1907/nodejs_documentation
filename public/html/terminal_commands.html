Terminal Commands


To start server/app:

node 'filename'


installs express local in projects folder with:

npm install express

express can also be installed by declaring it in the pacakage.json file manually and using the command

npm install

this command installs all the dependencies declared in the package.json file.


Nodemon commands

Installs it globally:

nodemon install -g nodemon

installs it related only to this project:

npm install --save-dev nodemon

To use nodemon:

nodemon 'filename'
